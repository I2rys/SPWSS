## Installation
Github:

    git clone https://github.com/I2rys/SPWSS
    
NPM Packages:
```
npm i request
```

## Usage
```
node index.js <domain> <output>
```

+ domain - The website domain to enumerate.
+ output - The output of the found subdomains in the domain.
